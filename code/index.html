Sticker

iOS and Web Development

Sticker is an app that aggregates professional reviews reviews for restaurants. In short, it is the Metacritic for the restaurant industry. This project is currently in development.

In addition to building the iPhone app, I also built a robust admin interface for editors to add content such as reviews, restaurant photos, and awards

UO Web Dev Celebrememe Tutorial

Web Development

I wanted to introduce the University Of Oregon Web Development Club to server side web development and working with API's in a fun, approachable way. So I created a Flask app where a user enters a celebrity and their last tweet is turned into a meme.
